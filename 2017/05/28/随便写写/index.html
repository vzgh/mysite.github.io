随便写写

发表于 2017-05-28

我的日常停更很久了，这是我不曾打算的也是我曾预料到的，并不是因为忙到没时间更，也不是因为没事发生没得更，算了算日子已经有三周了，可是说实话来了深圳的两个月都没怎么用心地更过，那不妨写一篇新的，扯扯淡说说话。

今天和魏广举去了广州，之前老爸一直说，来了深圳有机会去广州、珠海、东莞看看，也一直打算但一直没去，说起来原因可能有两个，一个是懒，一个是考虑到经费问题，好吧，第二个是我编的，我连预算都没写过。这次也多亏举哥，本来28号有人来调系统，突然不来了，举哥就说把广州之行提前一两天，其实27号晚上举哥给我提议的时候我内心是拒绝的，但是，这是我的老毛病了，也是之前很多行程泡汤的原因，我懂，这次强迫自己没有拒绝，也确实不好拒绝。为啥不好拒绝呢，涉及别人，我不想在我的博客里谈论对别人的看法。

昨天跟爸妈视频说到了对一个朋友的看法，我对这个人的很多行为很是看不上，我爸眉头一皱觉得事情并不是那么简单，他让我不要以自己的成长环境来揣测别人的想法。我之前把我的这种对待人的态度叫做爱恨分明，觉得一个人哪有那么多弯弯绕，喜欢就是喜欢，不喜欢就是怼你，可是我为什么会不喜欢呢？因为不喜欢别人为人处世的风格，觉得不是一类人？其实每个人这么做都有自己的苦衷吧，可能是他们内心最不愿意被触及的地方？当然了这都是我的猜测。

来了深圳之后想继续看《穆斯林的葬礼》，可是中间隔了一个月发现前面的有些想不起来了，每次拿起来看两段就又放下。经常在朋友圈看到晒书，晒出去旅行的，我觉得这是我喜欢的生活，我羡慕刘禹铭的生活，觉得他活的洒脱、充实、满足，但是我并不知道他付出了什么来获得这样的生活，也不知道为什么自己这样没有动力。

说起我向往的生活，除了记得和耿曼姐姐提起过也想不起谁了，但是肯定有，我喜欢到处宣扬我的理想因为我觉得这能提升我在别人心目中的逼格。在一个地方打工，挣够了去下一个地方的钱就前往下一个目的地，不会因为什么停留，从家往西走，从东边回家，这个要多久呢？一年？三年？十年？我不知道，那次和姐姐说，她说这挺需要魄力的，我也这么觉得，而且觉得自己恰恰没这个魄力，觉得这可能是我永远无法完成的理想。每次在电影里看到这样的人我总是能大哭一场。

来深圳两个月了，做了什么呢？一直调系统？哪有这么忙，玩游戏？也没本科的时候那样玩啊，看论文？四篇够少了吧，都没脸找老师汇报，学JAVA？你TM在逗我…自己的新年计划进行了多少呢？今天刚去了中山大学，又激起了我学摄影的想法，还是老问题，先学还是先买机器，买机器是一步到位还是循序渐进，没错，我是个墨迹的人，不太想找老爸要钱，觉得这么多年要了不少钱了，自己也二十好几了，可是自己又出不出来，昨天视频说到了以后工作买房，听我爸的意思是我确定了城市他帮我买，可是我想自己按揭，觉得爸妈奋斗了一辈子，挣得这些钱就该自己“挥霍”了，我没有权利找他们要这要那，当然了我这话说的有点太好听了，说的好像自己要的少似的。

今天先写到这吧，明天看看相机，看看论文，学学JAVA，列个时间表，看看能不能假期期间把机器买了。
